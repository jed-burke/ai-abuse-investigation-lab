# Case 007 Playbook

This case is about content features plus account context.

## Step 1: Count Features

- Count social-engineering categories.
- Count policy signals by account.
- Identify repeated template patterns.

## Step 2: Review Context

- Join account metadata.
- Compare established training account to related free account.
- Look for shared infrastructure.

## Step 3: Judge the Split

- Which activity looks authorized?
- Which activity looks suspicious?
- Should the same decision apply to both accounts?

## Step 4: Recommend Action

Use split outcomes if the evidence differs by account.
