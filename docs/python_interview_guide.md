# Python Interview Guide

Use the synthetic CSV to practice practical Python data work.

## Skills To Practice

- reading CSV files
- dictionaries, lists, and counters
- grouping records by account or session
- parsing timestamps
- calculating averages and thresholds
- writing small, testable functions
- explaining tradeoffs clearly

## Practice Pattern

1. Load the data.
2. Validate the fields you expect.
3. Compute a small summary.
4. Print or return a clear result.
5. Refactor repeated logic into a function.
