# SQL Interview Guide

Use the lab database to practice common interview patterns.

## Skills To Practice

- `SELECT`, `WHERE`, `ORDER BY`, and `LIMIT`
- `COUNT`, `AVG`, `MIN`, `MAX`, and `GROUP BY`
- `CASE` expressions for bucketing risk
- `JOIN` between `ai_usage_logs` and `account_profiles`
- CTEs with `WITH`
- window functions such as `ROW_NUMBER`, `LAG`, and rolling counts
- time filtering with ISO timestamps

## Practice Pattern

1. Write the simplest query that answers the question.
2. Add readable aliases.
3. Check row counts before trusting joins.
4. Explain the result in plain English.
5. Name one limitation of the query.
